# ContentXForm

Code and analysis for the paper in [contentVsForm.pdf](contentVsForm.pdf).

## About This Project

This repository contains the experimental pipeline used in the paper, including:

- Data preparation and preprocessing
- Scoring and descriptive analyses
- Decomposition and robustness analyses
- Rewriting workflows and downstream evaluation
- Figure, embedding, model, and results generation

The implementation is organized around Jupyter notebooks for each stage of the workflow, plus supporting resources and output folders.

## Paper

The paper PDF is available at:

- [contentVsForm.pdf](contentVsForm.pdf)

## Repository Structure

- [data](data): Raw and processed datasets
- [notebooks](notebooks): End-to-end analysis notebooks
- [outputs](outputs): Generated artifacts (figures, embeddings, models, rewrites, and results)
- [requirements.txt](requirements.txt): Python dependencies

## Quick Start

1. Create and activate a Python environment.
2. Install dependencies:

```bash
pip install -r requirements.txt
```

3. Open and run the notebooks in sequence based on your analysis needs.

## Notes

- Some notebooks may depend on intermediate artifacts generated by earlier notebooks.
- Large outputs are organized under [outputs](outputs) by artifact type.

## Notebook Summary

- [notebooks/00_setup.ipynb](notebooks/00_setup.ipynb): Environment and project setup, including initial paths, resources, and helper configuration.
- [notebooks/01_preprocessing.ipynb](notebooks/01_preprocessing.ipynb): Data cleaning and preprocessing pipeline for downstream modeling and analysis.
- [notebooks/2.0_hyperparam_tuning.ipynb](notebooks/2.0_hyperparam_tuning.ipynb): Hyperparameter search and tuning experiments.
- [notebooks/2.1_low_scorer.ipynb](notebooks/2.1_low_scorer.ipynb): Focused analysis/model behavior for low-scoring samples.
- [notebooks/2.2_high_scorer.ipynb](notebooks/2.2_high_scorer.ipynb): Focused analysis/model behavior for high-scoring samples.
- [notebooks/3.0_rewriter.ipynb](notebooks/3.0_rewriter.ipynb): Main rewriting pipeline for transforming text outputs.
- [notebooks/3.1_rewrite_preproc.ipynb](notebooks/3.1_rewrite_preproc.ipynb): Preprocessing steps specific to rewrite-generation or rewrite evaluation.
- [notebooks/04_scorer.ipynb](notebooks/04_scorer.ipynb): Core scoring workflow used to evaluate text samples.
- [notebooks/05_descriptive.ipynb](notebooks/05_descriptive.ipynb): Descriptive/statistical exploration of processed data and score distributions.
- [notebooks/06_decomp_bootstraping.ipynb](notebooks/06_decomp_bootstraping.ipynb): Bootstrap-based decomposition analysis to estimate component effects and uncertainty.
- [notebooks/6.1_robustness_decomp.ipynb](notebooks/6.1_robustness_decomp.ipynb): Robustness checks for decomposition results under alternative settings.
- [notebooks/07_measuring_redundency.ipynb](notebooks/07_measuring_redundency.ipynb): Analysis focused on measuring redundancy across features/outputs.
- [notebooks/08_images.ipynb](notebooks/08_images.ipynb): Figure and image generation for reporting/visualization.
